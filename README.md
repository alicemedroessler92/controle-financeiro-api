# 💰 Controle Financeiro API

API REST para gerenciamento de receitas e despesas.

## 🚀 Tecnologias
- Node.js
- JavaScript
- Express

## 📌 Funcionalidades
- Cadastro de receitas
- Cadastro de despesas
- Listagem de transações
- Cálculo de saldo

## ▶️ Como rodar
1. Clone o repositório
2. npm install
3. npm run dev
